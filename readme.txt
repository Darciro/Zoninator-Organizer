A Wordpress plugin that allows organization of posts into zones on publication

== Description ==

A Wordpress plugin that allows organization of posts into zones on publication. Ready for translation, already in english and portuguese (Brazil).

This plugin depends on the [Zone Manager (Zoninator)](http://wordpress.org/plugins/zoninator/) plugin and extends it's functionality by allowing you to specify the zone and position upon post creation/edition page, easily by drag and drop.

== Installation ==

1. Unzip the contents from this plugin onto `/wp-content/plugins/` directory;
1. Activate the plugin through the 'Plugins' menu in WordPress;
1. On the create (or edit) post page, add the post to a zone, and organize by drag and drop;
1. Enjoy!
